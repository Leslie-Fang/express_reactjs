基于express和reactjs搭建的一个最小功能系统

可以在这个的基础上进行业务需求的开发

目前已经包含的模块：
1.后端：express
2.数据库：mysql，以后进一步添加mongodb和redis的接口
3.前端：reatjs+redux
4.UI组件使用bootstrap

还来不及开发的部分
1.数据库接口
2.redux中的action部分
以上两点可以通过开发一个用户登入界面来实现

usage:

For production
1. git clone
2. npm install --production

For developer
1. git clone
2. npm install
